This project is a simple task tracker where you can add new task, delete an existing task, see what tasks you have added so far and mark a task as read.

Key features:
1. Add task
2. View tasks
3. Delete task
4. Mark a task as read
4. "task.json" to store the tasks and load the existing tasks while starting the program
5. "taskTracker.bat" file for introducing "double-click and run the program" feature
